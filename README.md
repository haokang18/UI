# 穿越火线官网
- web2.0风格，简单的设计
- 导航栏分2个，一个在界面右侧，与常规的不同，运用了级联菜单；一个在页面上部
- 首页分版面，设计较为统一
- 用空间来间隔各区块而不是分隔符
- 黑白色系，各颜色混用
- 交互性，用户可以进行评论留言
# 京东商城
- 导航栏在页面左部，banner在页面中部，符合格局
- 种类繁杂，分类不够仔细，页面下部信息用户往往浏览不到
- 易用性较好，搜索功能较为强大
- Web2.0风格，logo明显，设计简单
